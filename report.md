### Git
  Git is a version control software used to track your code.
  
  Git commands we used in this lab :
    - git init
    - git add <filename>
    - git commit -m "commit message"
    - git push
    - git branch <new-branch-name>
    - git checkout 
    - git merge
    - git clone

### Bash
  It is shell language
    
  commands we used :
      - pwd (print working directory)
      - touch (create new file)
      - cat (print file content to terminal)
      - rm (remove/delete file)
